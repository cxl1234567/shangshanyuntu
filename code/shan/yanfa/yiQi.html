PROJECT  DISPLAY

风机监控系统

风能是可再生的清洁能源，储量大、分布广，为了更好的开发利用风能。本项目实现对风力发电机各种 参数与指标的实时统计与监测。 通过对风机的各项指标的监测，将风机传回的所有数据，通过计算转换成直观详尽的统计图便于技术人 员进行分析对比，凭此对风力发电进行调整。从而达到最优指标，提高风能的利用率。另一方面，本项 目通过对风机状态的实时监控，能够第一时间监测到风机的异常状况并且将消息反馈给相关技术人员。 及时解决风机的一切异常问题，保证风机的正常运行。
