PROJECT  INDRODUCE

贵州燃气抄表系统

为节省人力成本，使燃气数据的录入与管理更加方便快捷。本项目通过配合抄表手持器，实现自动抄表录入数据并进行统计 形成报表。通过报表的直观展示，使工作人员能够更清晰精确的对数据进行分析。在本项目中对大量的抄表数据进行统一管 理，工作人员能够通过系统中统计的，以抄表数据、未抄表数据以及警告数据制定更加高效的工作计划。 工作人员能够在第一时间对反馈回来的信息进行针对性处理。在节省人力的同时提高了抄表工作的效率和准确度。
