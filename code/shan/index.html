WHY CHOOSE SHANGSHAN CLOUD ATLAS

上善云图隶属于中橙集团，是以高精尖技术为依托的科技公司。集团公司目前在职人员500人，办公场地超过2万平米，控股10家子公司，年产值超过1.5亿元。公司专注于企业信息化领域建设，致力于企业信息技术应用一体化方案、产品及服务的研究。
